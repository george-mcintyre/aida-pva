Detailed Description

The AidaService class is the main class that gets run to start an AIDA-PVA Provider.

It automatically loads the Native Provider Library on startup.

It also runs an AidaChannelProvider listener so that it can intercept search requests for channels that the loaded Native Channel Provider supports and direct them to the correct entry points.
