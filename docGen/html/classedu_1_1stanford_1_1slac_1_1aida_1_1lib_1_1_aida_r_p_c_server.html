void edu.stanford.slac.aida.lib.AidaRPCServer.registerServices ( AidaRPCService  service )

Register all channels in the given service This works by asking the channel provider for the channels that it supports.

The channel provider will load the supported channels from its CHANNEL.YML file if they are not already loaded. Once we have the list of channel names we pass them to the EPICs subsystem to register them so that the RPCServer will listen out for requests on our behalf.

service the given service
