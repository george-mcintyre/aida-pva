Detailed Description

This class encapsulates an AidaArguments structure.

It is used to represent all argument passed in an AIDA-PVA request.

Note that it uses the lombok.Data annotation to provide all the getters and setters. a constructor with all required arguments, and an equals(), hashcode() and toString() method.
