Detailed Description

This class encapsulates an AidaChannelConfig.

The AidaChannelConfig class controls how the AIDA-PVA Provider responds to a channel get or set request. There is a different AidaChannelConfig for get requests and set requests.

All set requests are differentiated by containing a VALUE AidaArgument - that is one with the name = VALUE.

The default AidaChannelConfig for set requests is NONE meaning that no set are allowed. To enable set requests an AidaChannelConfig of AidaChannelConfig#type = AidaType#NONE, or AidaChannelConfig#type = AidaType#TABLE must be set.

Note that it uses the lombok.Data annotation to provide all the getters and setters. a constructor with all required arguments, and an equals(), hashcode() and toString() method.

It also uses the lombok.NoArgsConstructor annotation to provide a constructor with no arguments.
