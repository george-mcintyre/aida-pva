Detailed Description

This class encapsulates an AidaConfigGroup.

It is used to represent the configuration that will apply to a single group of AidaChannels.

The AidaConfigGroup class controls how the AIDA-PVA Channel Provider responds to a channel request.

There is a AidaConfigGroup#getterConfig for the get requests, and a AidaConfigGroup#setterConfig for the set requests.

The List of AidaConfigGroup#channels are the names of the channels that will use the specified config

Note that it uses the lombok.Data annotation to provide all the getters and setters. a constructor with all required arguments, and an equals(), hashcode() and toString() method.

It also uses the lombok.NoArgsConstructor annotation to provide a constructor with no arguments.
