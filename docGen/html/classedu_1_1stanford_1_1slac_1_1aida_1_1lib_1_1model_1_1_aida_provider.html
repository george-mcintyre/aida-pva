Detailed Description

This class encapsulates an AidaProvider.

It is used configure how an AIDA-PVA will operate.

It has an AidaProvider#id to identify the Provider. It has an AidaProvider#name to give it a name. It has a AidaProvider#description displayed during startup alongside the AidaProvider#id and AidaProvider#name. The AidaProvider#configurations define the different AidaConfigGroup we define for requests to this channel. Finally, the AidaProvider#channelProvider defines the actual class that will implement the functionality to service this channel.

Note that it uses the lombok.Data annotation to provide all the getters and setters. a constructor with all required arguments, and an equals(), hashcode() and toString() method.

It also uses the lombok.NoArgsConstructor annotation to provide a constructor with no arguments.
