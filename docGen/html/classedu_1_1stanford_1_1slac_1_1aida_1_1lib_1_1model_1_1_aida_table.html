Detailed Description

This class encapsulates an AidaTable.

It is the class that is returned from the Native Providers for requests that return a TABLE.

It contains a single property AidaTable#data that stores a List of column Lists.

Note that it uses the lombok.ToString annotation to provide the toString() method.

Object  object

Add an element to the specified column in this AidaTable#data.

Each new column is added by successive calls to AidaTable#add and so the columnId specified will either have to be inserted, or added at the end.

This is called by the Native Channel Provider code in C so be careful if refactoring the signature or name.

Parameters
columnIdthe number representing the number of the column to add to
objectthe object to add to that column
Returns
true if added correctly
