Detailed Description

Unsupported Channel Type Exception.

Use this when the channel requested is registered with this service but a required TYPE parameter is missing or incorrect.

1) The channel is supported but requires a specific set TYPE parameters, one of which was not specified with the request 2) The TYPE that is sent with the request is not supported 3) etc
