Detailed Description

Represents a floating point number.

This can be either a single or double precision value. The isFloat flag is set to true if this is a single precision value and the value can be accessed via the value.floatValue. If the isFloat flag is set to false then this is a double precision value and the value can be accessed via the value.doubleValue.
